# Super Alveyworld
The sequel to the wildly successful Alveyworld: The Video Game.

## Starter kit
This game will be based on the excellent [starter kit](https://github.com/alveyworld-dev/starterkit) found yonder.  Go ahead and download it, play around, and read the README.

## Teams
For the development of this game, everyone will need to be involved.  The class will be split into several teams based on expertise.
* Coding Team
* Art Team
* Audio Team
